# Radio Player — Enhanced

Как запустить:

1. Установи Node.js (v18+).
2. В папке проекта выполни:

```
npm install
npm run dev
```

3. Открой http://localhost:5173

Если увидишь ошибку про @vitejs/plugin-react — выполни `npm install` ещё раз.

CORS: некоторые потоки блокируют прямые запросы из браузера. Для таких случаев используй proxy (поле в UI) или серверный прокси.
